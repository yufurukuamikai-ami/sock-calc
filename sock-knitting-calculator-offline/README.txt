靴下編み計算機（オフライン版）

使い方
1) sock-knitting-calculator-offline.html をダブルクリックして開きます。
2) 数値や選択肢を入れて「計算する」を押すだけです。

補足
- このHTMLは単体で動くので、ネット接続は不要です。
- iPhoneの場合：ファイルAppからHTMLを開いてSafariで表示→共有→「ホーム画面に追加」で“アプリっぽく”使えます（環境によって表示が変わることがあります）。

作成者向け
- もしWeb公開してログイン制を付けたい場合は、Cloudflare Access / Basic認証 / 会員サイト化などに拡張できます。
